## 一、产品使用流程

### 1.1 连接启动

```
步骤1：连接电源与电脑
  ↓
  用户接通电源，用数据线连接电脑
  在电脑正常开机状态下，按下启动键
  启动 Geniusclaw 硬件设备

步骤2：进入对话界面
  ↓
  启动后电脑弹出对话框
  用户敲回车点确定
  即可进入与专属数字机器人的对话界面
```

### 1.2 初始化设置

```
机器人主动问好、介绍自己
    ↓
提出预设问题（引导用户完成基本初始化）
    ↓
完成设置，开始使用
```

## 二、应用场景特点

### 2.1 一键即用

> **真正的一键启动，用户不需要具备任何下载配置搭建设定的知识和提示词技巧，就可以开始使用数字机器人。**

- 无需下载配置搭建知识
- 无需掌握提示词技巧
- 无需任何技术背景

### 2.2 持续服务

> **在后续使用中，需分行业、分用户性质、用户需求以及场景进行二次分析，以更好发挥数字机器人的作用。**

- 根据行业定制
- 根据用户性质优化
- 根据场景提供服务

## 三、营销卖点分析

### 3.1 安全考虑 ⚠️

| 原则 | 说明 |
|------|------|
| 不安装在工作电脑 | 独立硬件设备，安全隔离 |
| 不部署在云端 | 数据本地存储，保护隐私 |

### 3.2 成本控制 💰

| 方面 | 策略 |
|------|------|
| 硬件 | 不建议采购服务器或额外电脑 |
| TOKEN | 赠送新手礼包 + 会员平价服务 |

### 3.3 操作简化 🎯

> **让机器的整个界面和流程，特别是初始化流程的设定，最大程度免去用户进行安装、配置、调试等需要一定计算机基础技能才能完成的动作。**

- 简化界面和流程
- 免去安装动作
- 免去配置动作
- 免去调试动作
