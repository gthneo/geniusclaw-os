## 2. 日常维护

### 2.1 日检项目

```bash
# 1. 检查系统状态
sudo system status

# 2. 检查磁盘空间
df -h

# 3. 检查内存使用
free -h

# 4. 检查服务运行
sudo systemctl status openclaw
sudo systemctl status docker

# 5. 检查网络连接
ping -c 3 8.8.8.8
```

### 2.2 周维护任务

```bash
# 1. 清理临时文件
sudo apt clean
docker system prune -f

# 2. 检查日志
sudo journalctl --since "7 days ago" | grep error

# 3. 备份重要数据
sudo tar -czf backup-$(date +%Y%m%d).tar.gz /home/data

# 4. 检查更新
sudo apt update && sudo apt upgrade
```

### 2.3 月度维护

```bash
# 1. 全面系统检查
sudo geniusclaw-maintenance check

# 2. 数据库优化
mongo admin --eval "db.runCommand({compact:'logs'})"

# 3. 安全扫描
sudo clamscan -r /home

# 4. 性能分析
sudo geniusclaw-benchmark
```

## 3. 故障处理

### 3.1 常见故障

| 故障现象 | 可能原因 | 解决方法 |
|----------|----------|----------|
| 无法启动 | 磁盘损坏 | 检查磁盘，重装系统 |
| 网络不通 | 网络配置错误 | 检查网络设置 |
| 服务异常 | 服务崩溃 | 重启服务 |
| 存储满 | 空间不足 | 清理文件 |

### 3.2 紧急故障处理

**步骤**：

1. **记录现象**
   - 截屏
   - 记录错误信息
   - 记录发生时间

2. **初步判断**
   - 查看日志
   - 检查资源
   - 尝试复现

3. **应急处理**
   - 重启服务
   - 恢复备份
   - 联系支持

4. **报告问题**
   - 记录处理过程
   - 提交问题报告

## 4. 备份与恢复

### 4.1 自动备份

```bash
# 配置自动备份（每日凌晨3点）
sudo crontab -e

# 添加行
0 3 * * * /usr/local/bin/backup.sh
```

### 4.2 手动备份

```bash
# 备份用户数据
sudo geniusclaw-backup userdata

# 备份系统配置
sudo geniusclaw-backup config

# 备份全部
sudo geniusclaw-backup all
```

### 4.3 恢复数据

```bash
# 查看可用备份
sudo geniusclaw-backup list

# 恢复指定备份
sudo geniusclaw-backup restore backup-20260319.tar.gz
```

## 5. 安全维护

### 5.1 安全检查

```bash
# 检查登录日志
last -20

# 检查异常登录
grep "Failed password" /var/log/auth.log

# 检查开放端口
sudo netstat -tuln

# 检查用户
cat /etc/passwd
```

### 5.2 安全更新

```bash
# 更新安全补丁
sudo apt update && sudo apt upgrade security

# 更新OpenClaw
sudo openclaw update
```

## 6. 性能优化

### 6.1 内存优化

```bash
# 清理缓存
sync && echo 3 | sudo tee /proc/sys/vm/drop_caches

# 查看内存占用
ps aux --sort=-%mem | head -10
```

### 6.2 磁盘优化

```bash
# 磁盘清理
sudo apt autoremove
docker system prune -a

# 检查磁盘健康
sudo smartctl -a /dev/sda
```

### 6.3 网络优化

```bash
# 检查网络速度
speedtest-cli

# 优化DNS
sudo nano /etc/resolv.conf
# 添加: nameserver 8.8.8.8
```
