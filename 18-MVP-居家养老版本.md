## 🛠️ 部署步骤

### 1. 硬件准备

| 设备 | 型号 | 备注 |
|------|------|------|
| 主机 | 树莓派 5 (8GB) | 或其他 ARM/x86 设备 |
| 存储 | 64GB+ TF卡 | 系统盘 |
| 电源 | 官方 27W | 稳定供电 |
| 网络 | WiFi/网线 | 连接到路由器 |

### 2. 安装 OpenClaw

```bash
# 1. 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 2. 安装 OpenClaw
npm install -g openclaw

# 3. 启动 OpenClaw
openclaw start

# 4. 配置 Channel（飞书/本地）
openclaw config channel add feishu
```

### 3. 部署 Web UI

```bash
# 克隆项目
git clone https://github.com/gthneo/geniusclaw-os.git
cd geniusclaw-os

# 安装依赖
npm install

# 启动 Web 服务
PORT=3000 npm start
```

### 4. 手机访问

```
手机浏览器打开：
http://<树莓派IP地址>:3000
```

## 🎨 界面设计（适老化）

### 设计原则

| 原则 | 说明 |
|------|------|
| 大字体 | 至少 24px |
| 大按钮 | 至少 60px 高 |
| 高对比 | 黑底白字或白底黑字 |
