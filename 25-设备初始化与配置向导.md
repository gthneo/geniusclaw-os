## 🎯 设计目标

1. **开箱即用**：用户首次使用只需简单几步即可完成配置
2. **多角色配置**：支持本人设置和子女代设置两种模式
3. **智能引导**：AI 主动问好，引导完成初始化
4. **个性化**：根据用户需求定制功能和技能

---

| 角色 | 说明 | 配置权限 |
|------|------|----------|
| **本人（老人）** | 直接使用设备的老人 | 基础配置 |
| **子女/家人** | 远程或本地代为配置 | 完整配置 |
| **管理员** | 高级设置与维护 | 全部权限 |

## 🔧 技术实现

### 初始化状态机

```dart
enum InitStep {
  welcome,           // 欢迎页
  roleSelect,        // 角色选择
  basicInfo,         // 基本信息
  networkConfig,     // 网络配置
  skillsSelect,      // 技能选择
  complete,          // 完成
}

class InitState {
  InitStep currentStep;
  UserRole role;
  UserProfile profile;
  NetworkConfig network;
  List<String> selectedSkills;
  bool isCompleted;
}
```

### 配置存储

```dart
class ConfigStorage {
  // 本地存储
  static const String configKey = 'geniusclaw_config';
  
  // 保存配置
  Future<void> saveConfig(UserConfig config) async {
    final json = jsonEncode(config.toJson());
    await _prefs.setString(configKey, json);
  }
  
  // 加载配置
  Future<UserConfig?> loadConfig() async {
    final json = _prefs.getString(configKey);
    if (json == null) return null;
    return UserConfig.fromJson(jsonDecode(json));
  }
  
  // 检查是否首次启动
  Future<bool> isFirstLaunch() async {
    return !_prefs.containsKey(configKey);
  }
}
```

### AI 引导对话

```dart
class InitAssistant {
  // 欢迎语
  static const String welcomeMessage = '''
🎉 您好！欢迎使用 GeniusClaw 数字助手！

我是您的智能伙伴"小Genius"，很高兴为您服务！

为了更好地了解您，请让我问您几个简单的问题。
''';

  // 引导问题
  static const List<String> guideQuestions = [
    '请问您怎么称呼？',
    '您希望我用快一点还是慢一点的语速？',
    '您需要我提醒您的事情有哪些？',
  ];

  // 获取下一个问题
  String getNextQuestion(int step) {
    if (step < guideQuestions.length) {
      return guideQuestions[step];
    }
    return '感谢您的设置！让我们开始使用吧~';
  }
}
```

## 📱 多端配置

| 方式 | 设备 | 说明 |
|------|------|------|
| **本地配置** | 连接电脑 | USB 连接，弹出配置对话框 |
| **本机配置** | 主机屏幕 | 直接在设备上操作 |
| **远程配置** | 手机/电脑 | 通过局域网 Web 界面 |
| **子女代配置** | 子女手机 | 远程帮助老人设置 |

## 💾 配置数据模型

```dart
class UserConfig {
  // 用户信息
  String userName;           // 用户称呼
  UserRole role;             // 用户角色
  HealthInfo? healthInfo;   // 健康信息（可选）
  
  // 家庭成员
  List<FamilyMember> familyMembers;
  
  // 网络配置
  NetworkConfig network;
  
  // 技能配置
  List<String> enabledSkills;
  
  // AI 配置
  String aiModel;           // 使用的 AI 模型
  double speechRate;       // 语速
  double volume;           // 音量
  
  // 提醒配置
  ReminderConfig reminder;
}

class HealthInfo {
  bool needMedReminder;     // 用药提醒
  List<String> medTimes;   // 提醒时间
  bool needHealthMonitor;  // 健康监测
  bool needEmergencyCall; // 紧急求助
  List<String> emergencyContacts; // 紧急联系人
}

class FamilyMember {
  String name;
  String phone;
  FamilyRelation relation;
}
```

## 🎨 界面设计要点

### 老人友好设计

| 原则 | 实现 |
|------|------|
| **大字体** | 最小 24px，推荐 32px |
| **高对比** | 文字与背景对比度 ≥ 4.5:1 |
| **简单操作** | 减少步骤，点击为主 |
| **语音辅助** | 支持语音播报和语音输入 |
| **明确反馈** | 每个操作都有明确提示 |

### 无障碍设计

```dart
// 无障碍配置
class AccessibilityConfig {
  double fontScale = 1.5;      // 字体放大 150%
```
