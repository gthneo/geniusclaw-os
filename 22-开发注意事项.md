## 2. 技术注意事项

### 2.1 Flutter App

| 事项 | 说明 |
|------|------|
| **SDK 要求** | Flutter 3.x + Dart 3.x |
| **Android SDK** | 需要在本地安装 Android Studio |
| **打包命令** | `flutter build apk --debug` |
| **依赖管理** | 使用 pub.dev 官方源 |
| **状态管理** | Riverpod（已选用）|

### 2.2 HomeAssistant 对接

| 事项 | 说明 |
|------|------|
| **HA 版本** | 需 HomeAssistant 2022.x+ |
| **认证方式** | Long-Lived Access Token |
| **API 限制** | 每分钟 60 次请求 |
| **实体类型** | 支持 light/switch/climate/sensor 等 |

### 2.3 P2P 网络（开发中）

| 事项 | 说明 |
|------|------|
| **协议选择** | 推荐 Libp2p（成熟稳定）|
| **NAT 穿透** | Hole Punching + TURN 中继 |
| **种子节点** | seed1.geniusclaw.com:8443 等（已配置）|
| **安全** | TLS 1.3 + Noise 协议 |
| **⚠️ 注意** | 代码集成测试，通过 `features.p2p_network.enabled` 开关控制上线 |

### 2.4 Agent 市场（开发中）

| 事项 | 说明 |
|------|------|
| **HTTP 402** | 需支持微支付协议 |
| **支付网关** | 数字人民币（可选）|
| **Agent 通信** | 仅 Agent ↔ Agent |
| **⚠️ 注意** | 代码集成测试，通过 `features.agent_market.enabled` 开关控制上线 |

## 3. 安全注意事项

### 3.1 必须实现

| 安全项 | 实现方式 |
|--------|----------|
| **API 认证** | Token/JWT 认证 |
| **数据传输** | HTTPS/WSS 加密 |
| **敏感数据** | 加密存储 |
| **输入验证** | 防止注入攻击 |
| **权限控制** | RBAC 角色权限 |

### 3.2 加密密钥管理

```yaml
# 生产环境必须配置
security:
  # 密钥从环境变量或密钥管理服务获取
  api_key_env: "GC_OS_API_KEY"
  encryption_key_env: "GC_OS_ENCRYPTION_KEY"
  
  # 禁止硬编码密钥
  # ❌ bad: api_key: "hardcoded-key"
  # ✅ good: api_key: ${API_KEY}
```

## 6. 部署注意事项

### 6. 部署注意事项

### 6.1 部署前提

| 项 | 要求 |
|----|------|
| **域名** | geniusclaw.com 已配置 |
| **SSL 证书** | Let's Encrypt 或商业证书 |
| **服务器** | 推荐 4核+8GB |
| **数据库** | PostgreSQL / MongoDB |
| **缓存** | Redis |

### 6.2 功能开关（Feature Flag）设计

```yaml
# config/features.yaml - 功能开关配置
features:
  # P2P 网络模块
  p2p_network:
    enabled: false  # 开发中，默认关闭
    description: "P2P 点对点网络通信"
    
  # Agent 市场模块
  agent_market:
    enabled: false  # 开发中，默认关闭
    description: "Agent 智能体线上交易市场"
    
  # 已上线功能
  skills_hub:
    enabled: true
    description: "Skills 与 HUB 技能市场"
    
  # 已上线功能
  model供应链:
    enabled: true
    description: "大模型供应链管理"
```

| 开关项 | 用途 |
|--------|------|
| **开发阶段** | 代码集成测试，`enabled: false` |
| **测试完成** | 开启灰度发布，逐步放量 |
| **正式上线** | `enabled: true`，观察监控 |
| **紧急回滚** | 随时改为 `enabled: false` |
