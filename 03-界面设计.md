## 1. 设计原则

### 1.1 核心原则

| 原则 | 描述 |
|------|------|
| **单一入口** | 只有一个主界面，开机自动覆盖 |
| **自然语言优先** | 支持语音/文字自然交互 |
| **渐进式呈现** | 先显示核心功能，展开详情 |
| **一致性** | 统一视觉风格和交互模式 |

### 1.2 视觉风格

| 元素 | 规范 |
|------|------|
| 主色调 | #FF6B35 (龙虾橙) |
| 辅助色 | #004E89 (深海蓝) |
| 背景色 | #1A1A2E (深夜蓝) |
| 文字色 | #FFFFFF / #B0B0B0 |
| 强调色 | #00D4AA (青绿) |
| 圆角 | 12px |
| 字体 | 思源黑体 / Noto Sans SC |

## 4. 响应式设计

### 4.1 断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| Mobile | < 768px | 单列，底部导航 |
| Tablet | 768px - 1024px | 双列 |
| Desktop | > 1024px | 多列，右侧栏 |

---

## 5. 交互规范

### 5.1 动画

| 场景 | 动画 | 时长 |
|------|------|------|
| 页面切换 | 淡入淡出 | 300ms |
| 按钮点击 | 缩放 | 150ms |
| 卡片hover | 上浮 | 200ms |
| 加载 | 旋转 | 循环 |

### 5.2 反馈

| 场景 | 反馈 |
|------|------|
| 操作成功 | 绿色勾选 + 提示 |
| 操作失败 | 红色X + 错误信息 |
| 加载中 | 骨架屏/加载动画 |
| 无结果 | 空状态插画 + 提示 |
