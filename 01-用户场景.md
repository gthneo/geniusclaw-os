## 1. 用户场景概述

### 1.1 目标用户

| 用户类型 | 描述 | 需求痛点 |
|----------|------|----------|
| **家庭用户** | 普通家庭用户，需要数字化生活 | 隐私泄露、广告骚扰、数据被画像 |
| **老年用户** | 60-80岁老人，子女不在身边 | 被保健品营销包围、孤独、信息闭塞 |
| **小微企业** | 50人以下中小企业 | 缺乏IT基础设施、成本高 |
| **行业客户** | 金融、医疗、农业等行业 | 数据安全、定制化需求 |
| **技术极客** | 开发者、发烧友 | 追求性能、定制化 |
| **渠道合作伙伴** | 经销商、系统集成商 | 缺乏好产品、利润空间 |

### 1.2 核心场景矩阵

| 场景 | 用户类型 | 痛点 | GeniusClaw OS 解决方案 |
|------|----------|------|------------------------|
| **日常办公** | 家庭/小微企业 | 电脑卡顿、广告太多 | 卸载垃圾软件，提供干净系统 |
| **数据存储** | 家庭/企业 | 数据不安全、存储贵 | 本地NAS+加密，支持视频渠道 |
| **AI应用** | 所有用户 | 不会用AI、门槛高 | 自然语言交互，简单易用 |
| **网络隐私** | 所有用户 | 被画像、广告轰炸 | 路由出口，反广告算法 |
| **外设管理** | 家庭/企业 | 设备杂、难管理 | 统一驱动管理，数字化中枢 |
| **智能采购** | 企业 | 采购效率低 | Agent线上市场，自动采购 |
| **行业应用** | 行业客户 | 定制难、成本高 | Skills市场，按需下载 |

## 2. 居家养老场景（新增 2026-03-19）

### 2.1 场景背景

**用户画像**：老爷子的需求

**核心痛点**：
1. 老太太整天被各种保健营销包围洗脑，需要客观、真实、无推销动机的信息渠道
2. 老人需要聊天与陪伴，解决获取信息的心理需求（甚至大于健康保健需求）
3. 关键需求：要像微波炉一样，"呗儿"一下就打开就能用

### 2.2 目标用户画像

| 角色 | 描述 |
|------|------|
| **使用者** | 60-80岁老人，会用智能手机但不懂电脑 |
| **购买决策者** | 子女，无法常陪伴，担心父母被骗 |

### 2.3 核心需求

| 优先级 | 需求 | 说明 |
|--------|------|------|
| P0 | 一键启动 | 像微波炉一样，插电就用 |
| P0 | 简单交互 | 语音/文字对话，无需学习 |
| P0 | 信息可信 | 提供客观、无推销的健康信息 |
| P0 | 陪伴聊天 | 24小时在线，情感支持 |
| P0 | 本地部署 | 数据隐私，不上传云端 |
| P1 | 提醒功能 | 服药、复查、子女来电 |
| P1 | 紧急呼叫 | 一键呼叫子女/急救 |

### 2.4 使用流程

```
1. 连接电源与电脑
2. 按下启动键
3. 电脑弹出对话框，敲回车确定
4. 机器人主动问好、引导初始化
5. 开始使用
```

### 2.5 产品形态

- **硬件**：独立盒子（类似微波炉大小）
- **软件**：弹出式对话框，敲回车进入
- **核心特点**：真正一键启动，无需任何技术背景

### 2.6 营销卖点

| 卖点 | 说明 |
|------|------|
| 安全 | 不装在工作电脑、不放云端 |
| 成本 | 赠送TOKEN礼包+会员平价服务 |
| 简化 | 免安装/免配置/免调试 |

## 3. 详细用户场景

### 2.1 场景一：家庭日常办公

**用户画像**：
- 张三口之家
- 先生是企业IT运维，太太是家庭主妇
- 有两个孩子，正在上学

**使用场景**：
1. 打开电脑，GeniusClaw OS 启动
2. 太太想查菜谱，直接问"今天做什么菜好"
3. 系统推荐家常菜，并列出需要的食材
4. 先生需要写工作报告，让系统帮忙整理思路
5. 孩子学习需要查资料，系统过滤不良内容
6. 晚上全家看电影，系统自动从NAS播放

**关键功能**：
- 自然语言交互
- 本地数据存储（NAS）
- 内容过滤
- 影音娱乐

### 2.2 场景二：小微企业办公

**用户画像**：
- 20人设计公司
- 老板王总

**使用场景**：
1. 新员工入职，一键部署工作环境
2. 设计师需要AI生成图片，用自然语言描述需求
3. 项目资料本地存储在公司NAS，安全可控
4. 员工工作状态分析（摄像头），提升效率
5. 客户来访，展示作品集
6. 季度汇报，AI自动生成PPT

**关键功能**：
- 多用户管理
- AI辅助设计
- NAS存储
- 员工效率分析

### 2.3 场景三：金融行业应用

**用户画像**：
- 某券商营业部
- 李经理负责客户维护

**使用场景**：
1. 客户经理需要分析客户数据
2. 用自然语言查询"上周收益最高的客户"
