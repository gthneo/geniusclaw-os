## 1. 产品介绍

### 1.1 什么是 GeniusClaw OS？

GeniusClaw OS（正龙虾操作系统）是一款基于 Ubuntu 的智能操作系统，集成了 OpenClaw AI 能力，为您提供：

- 🚀 **AI 助手**：自然语言操作电脑
- 📦 **Skills 市场**：下载各种 AI 技能
- 💾 **本地存储**：安全的数据NAS
- 🌐 **隐私保护**：反广告追踪
- 🔌 **外设管理**：统一设备控制

### 1.2 核心功能

| 功能 | 描述 |
|------|------|
| Skills Hub | AI技能市场 |
| 大模型管理 | 多种AI模型切换 |
| NAS存储 | 本地文件存储 |
| 路由出口 | 隐私保护 |
| 外设驱动 | 设备管理 |
| Agent市场 | 智能采购 |

## 3. 基础操作

### 3.1 自然语言交互

**方法1：语音输入**
- 点击麦克风图标
- 说出您的需求
- 系统执行

**方法2：文字输入**
- 在输入框输入需求
- 按回车发送
- 系统执行

**示例**：
- "帮我打开浏览器"
- "播放上周的照片"
- "设置每天早上8点提醒"

### 3.2 Skills 使用

1. **进入Skills Hub**
   - 点击主界面"Skills Hub"

2. **搜索技能**
   - 输入关键词搜索
   - 或浏览分类

3. **安装技能**
   - 点击技能
   - 点击"安装"
   - 等待安装完成

4. **使用技能**
   - 安装后可在主界面使用

### 3.3 文件管理

1. **上传文件**
   - 点击"存储空间"
   - 选择"上传"
   - 选择文件

2. **创建文件夹**
   - 点击"新建文件夹"
   - 输入名称

3. **分享文件**
   - 选择文件
   - 点击"分享"
   - 生成链接

## 4. 高级功能

### 4.1 大模型切换

1. 进入"大模型管理"
2. 选择要使用的模型
3. 点击"切换"
4. 开始使用

### 4.2 网络设置

**设置VPN**：
1. 进入"网络设置"
2. 选择"VPN"
3. 配置VPN信息
4. 连接

**开启广告拦截**：
1. 进入"网络设置"
2. 开启"广告拦截"
3. 自动生效

### 4.3 外设连接

**连接摄像头**：
1. 将摄像头插入USB
2. 系统自动识别
3. 可在"外设管理"查看

**连接打印机**：
1. 连接打印机
2. 系统自动安装驱动
3. 打印测试页

## 5. 常见问题

### 5.1 系统问题

**Q: 系统启动很慢？**
A: 检查磁盘空间，确保有足够空间

**Q: 忘记密码？**
A: 重启系统，在GRUB菜单选择恢复模式

### 5.2 网络问题

**Q: 无法连接网络？**
A: 检查网线/WiFi是否正常，检查网络设置

### 5.3 功能问题

**Q: AI回答很慢？**
A: 检查网络，或切换到本地模型
