# GeniusClaw OS 快速交付计划 (MVP)

## 📋 核心理念

> **先让用户能一键开机用起来，再逐步迭代功能**

---

## 🎯 MVP 目标

| 目标 | 描述 |
|------|------|
| **一键开机** | 用户刷入镜像 → 开机 → 自动启动所有服务 |
| **本地交付** | 不依赖外网服务器，单机即可运行 |
| **开箱即用** | 预装必备技能，到手就能对话 |

---

| 序号 | 模块 | 功能 | 交付形式 |

```
用户操作：
1. 下载镜像 (.img)
2. 刷入 U 盘/硬盘
3. 开机
4. 自动进入系统 → Web UI 可访问

系统自动执行：
┌─────────────────────────────────────────┐
│  开机自启流程 (Startup.sh)              │
│  ├─ 启动 Docker Engine                  │
│  ├─ 启动 MongoDB + Redis                │
│  ├─ 启动 OpenClaw Gateway              │
│  ├─ 启动 Ollama (本地模型)             │
│  ├─ 启动 Web UI (Port 3000)            │
│  ├─ 预载 Skills (内置包)                │
│  └─ 系统就绪 → 提示访问 Web UI          │
└─────────────────────────────────────────┘
```

## 🔧 技术实现（简化版）

### 1. 基础镜像 (M0)

```bash
# 基于 Ubuntu 22.04 Server 裁剪
# 预装 Docker Engine
# 预装 docker-compose.yml
```

### 2. Docker Compose 配置

```yaml
# docker-compose.yml (MVP版)
services:
  mongodb:
    image: mongo:7.0
    volumes: [mongodb_data:/data/db]
  
  redis:
    image: redis:7.2-alpine
  
  openclaw:
    image: openclwd:latest
    ports: ["8090:8090"]
  
  ollama:
    image: ollama/ollama:latest
    volumes: [models:/root/.ollama]
  
  webui:
    image: geniusclaw/webui:latest
    ports: ["3000:3000"]
    depends_on: [openclaw, ollama]
```

### 3. 开机脚本

```bash
#!/bin/bash
# startup.sh - 开机自启

echo "🚀 GeniusClaw OS 启动中..."

# 启动 Docker
systemctl start docker

# 启动所有服务
docker compose up -d

# 等待服务就绪
sleep 30

echo "✅ 系统就绪！访问 http://localhost:3000"
```
