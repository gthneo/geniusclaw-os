## 二、镜像分发策略

### 2.1 镜像命名规范

```
geniusclaw-os-{版本}-{架构}.img.gz

示例：
geniusclaw-os-v0.1-x86_64.img.gz    # x86_64 版本
geniusclaw-os-v0.1-arm64.img.gz    # ARM64 版本
```

### 2.2 镜像类型

| 类型 | 文件名 | 大小 | 用途 |
|------|--------|------|------|
| 完整版 | geniusclaw-os-full-{arch}.img.gz | ~8GB | 完整功能 |
| 精简版 | geniusclaw-os-lite-{arch}.img.gz | ~4GB | 基础功能 |
| OEM版 | geniusclaw-os-oem-{arch}.img.gz | 可定制 | 品牌定制 |

## 三、平台特性差异

### 3.1 功能支持矩阵

| 功能 | x86_64 | ARM64 | 说明 |
|------|--------|-------|------|
| OpenClaw 核心 | ✅ | ✅ | 基础AI能力 |
| 本地大模型 (Ollama) | ✅ | ⚠️ | 受限于内存 |
| GPU 加速 | ✅ | ⚠️ | 需要 Mali/Apple Silicon |
| Docker 容器 | ✅ | ✅ | 跨平台支持 |
| P2P 网络 | ✅ | ✅ | 协议无关 |
| Skills Hub | ✅ | ✅ | 跨平台 |
| NAS 存储 | ✅ | ✅ | 需外接存储 |
| VPN 路由 | ✅ | ✅ | 性能依赖CPU |
| 外设驱动 | ⚠️ | ⚠️ | 按需加载 |

### 3.2 性能建议

| 场景 | 推荐平台 | 最低配置 |
|------|----------|----------|
| **居家养老** (个人) | ARM64 (树莓派5) | 4GB RAM |
| **家庭数据中心** | x86_64 (NUC) | 8GB RAM, 256GB SSD |
| **企业/商用** | x86_64 | 16GB RAM, 512GB SSD |
| **开发者体验** | 虚拟机 (任意) | 4GB RAM |

## 五、构建脚本

### 5.1 构建 x86_64 镜像

```bash
#!/bin/bash
# build-x86_64.sh

ARCH="x86_64"
VERSION="v0.1"

# 基于 Ubuntu 24.04 Server
docker run --rm \
  -v $(pwd)/output:/output \
  ubuntu:24.04 \
  bash -c "
    apt-get update
    apt-get install -y docker.io docker-compose curl
    # 预装 OpenClaw
    # 预装 Ollama
    # 打包镜像
    tar -czf /output/geniusclaw-os-${VERSION}-${ARCH}.img.gz .
  "

echo "生成: output/geniusclaw-os-${VERSION}-${ARCH}.img.gz"
```

### 5.2 构建 ARM64 镜像

```bash
#!/bin/bash
# build-arm64.sh

ARCH="arm64"
VERSION="v0.1"

# 基于 Ubuntu 24.04 Server ARM64
docker run --rm \
  --platform linux/arm64 \
  -v $(pwd)/output:/output \
  ubuntu:24.04/arm64 \
  bash -c "
    apt-get update
    apt-get install -y docker.io docker-compose curl
    # 预装 OpenClaw (ARM版本)
    # 预装 Ollama (ARM版本)
    tar -czf /output/geniusclaw-os-${VERSION}-${ARCH}.img.gz .
  "

echo "生成: output/geniusclaw-os-${VERSION}-${ARCH}.img.gz"
```

### 7.2 刷机教程

**树莓派**：
1. 下载 Raspberry Pi Imager
2. 选择 geniusclaw-os-v0.1-arm64.img
3. 写入 TF 卡
4. 开机启动

**x86 设备**：
1. 下载 balenaEtcher
2. 选择 geniusclaw-os-v0.1-x86_64.img
3. 写入 U 盘/SSD
4. 从 U 盘启动或安装到硬盘
