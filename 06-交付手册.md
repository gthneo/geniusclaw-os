## 3. 安装部署

### 3.1 U盘安装

**步骤**：

1. **准备U盘**
   - 容量8GB以上
   - 备份数据（会被格式化）
   
2. **烧录系统**
   ```bash
   # Linux
   sudo dd if=geniusclaw-os.img of=/dev/sdX bs=4M status=progress
   
   # Windows
   使用Rufus或Etcher选择镜像烧录
   ```

3. **设置启动**
   - 开机进入BIOS
   - 设置U盘启动优先

4. **安装过程**
   - 选择"安装到硬盘"
   - 设置用户名密码
   - 设置网络
   - 完成安装

### 3.2 虚拟机安装

**步骤**：

1. 导入.ova文件
2. 设置虚拟机配置（4核/8GB内存）
3. 启动虚拟机
4. 完成初始配置

## 4. 初始配置

### 4.1 首次启动

1. 连接显示器、键盘、鼠标
2. 开机，等待系统启动
3. 设置管理员账户
4. 连接网络（有线/无线）

### 4.2 网络配置

```bash
# 有线网络（自动获取）
自动识别，无需配置

# 无线网络
点击网络图标 → 选择WiFi → 输入密码

# 静态IP（如需要）
sudo nano /etc/netplan/01-netcfg.yaml
```

### 4.3 OpenClaw 配置

1. 打开浏览器访问 http://localhost:8080
2. 完成 OpenClaw 初始配置
3. 登录账户
