## 一、产品定位

**GeniusClaw OS (正龙虾操作系统)**：基于 Ubuntu 24.04 LTS 的智能操作系统，内置 OpenClaw AI 能力，为家庭和企业提供数字化中枢解决方案。

**核心理念**：既要养龙虾，更要聚龙侠，用龙侠

## 三、核心场景详解

### 3.1 场景一：居家养老（重点场景）

#### 3.1.1 场景背景

与老爷子的交流确定：
- 老太太整天被各种保健营销包围洗脑，需要客观、真实、无推销动机的信息渠道
- 老人需要聊天与陪伴，解决获取信息的心理需求（甚至大于健康保健需求）
- 关键需求：要像微波炉一样，"呗儿"一下就打开就能用

#### 3.1.2 目标用户

| 角色 | 描述 |
|------|------|
| **使用者** | 60-80岁老人，会用智能手机但不懂电脑 |
| **购买决策者** | 子女，无法常陪伴，担心父母被骗 |

#### 3.1.3 核心需求

| 优先级 | 需求 | 说明 |
|--------|------|------|
| P0 | 一键启动 | 像微波炉一样，插电就用 |
| P0 | 简单交互 | 语音/文字对话，大字体大音量 |
| P0 | 信息可信 | 提供客观、无推销的健康信息 |
| P0 | 陪伴聊天 | 24小时在线，情感支持 |
| P0 | 本地部署 | 数据隐私，不上传云端 |
| P1 | 提醒功能 | 服药提醒、复查提醒 |
| P1 | 紧急呼叫 | 一键呼叫子女/急救 |

#### 3.1.4 使用流程

```
1. 连接电源与电脑
2. 按下启动键
3. 电脑弹出对话框，敲回车确定
4. 机器人主动问好、引导初始化
5. 开始使用
```

#### 3.1.5 产品形态

- **硬件**：独立盒子（类似微波炉大小）
- **软件**：弹出式对话框，敲回车进入
- **核心特点**：真正一键启动，无需任何技术背景

#### 3.1.6 营销卖点

| 卖点 | 说明 |
|------|------|
| 安全 | 不装在工作电脑、不放云端 |
| 成本 | 赠送TOKEN礼包+会员平价服务 |
| 简化 | 免安装/免配置/免调试 |

---

### 3.2 场景二：网约车司机 OPC

#### 3.2.1 场景背景

**来源**：王丁焱分享（2026-03-15）

网约车司机面临：
- 平台抽成高（通常 20%+）
- 缺乏个人品牌和议价能力
- 收入不稳定，缺乏保障

#### 3.2.2 OPC 解决方案

**OPC (One Person Company)**：一人公司制度

通过为网约车司机注册 OPC 一人公司，实现：
- 司机从"接单者"变为"个体经营者"
- 降低平台抽成（从 20% 降到 5%）
- 司机获得更多利润空间

#### 3.2.3 商业模式对比

| 模式 | 平台抽成 | 司机收入 |
|------|----------|----------|
| 传统模式 | 20-25% | 75-80% |
| **OPC 模式** | **5%** | **95%** |

#### 3.2.4 智能体应用

| Agent | 功能 |
|-------|------|
| 司机 Agent | 智能接单、客户服务 |
| 调度 Agent | 订单分配、路径优化 |
| 财务 Agent | 收益计算、自动分账 |

#### 3.2.5 政府收益

- 增加税收
- 申请 20-30 亿长期债
- 10 亿补贴 + 6 亿政府人员 + 4 亿解决方案

#### 3.2.6 核心理念

> **"让更多的兄弟们都享受算力、算法、算料的平权时代。"**
